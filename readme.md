# Mumu bird

## SNS app with lots of features
- Write a post
- Like/Unlike a post 
- Write a comment
- retweet
- infinite scrolling 
- Follow/Unfollow
- Search by hash tag, user 

## Technologies I used
- Next.js(SSR) 
- React Hooks 
- Redux 
- Redux saga 
- Ant Design
- Styled Components
- Node 
- Express 
- Sequelize(MySQL)
